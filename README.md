In cloud environments he prioritises predictable scaling, well-defined risk boundaries, and alerts that trigger immediate action when drift occurs.

Additionally, he contributes to Apprentice Systems, creating open-source Go tools that make SPF, DKIM, and DMARC straightforward for developers to implement and verify.
